#### run {#1-1}

> docker run命令是创建一个新容器并运行，若本地不存在该容器镜像会直接去镜像仓库中查找并自动下载。

**语法**

```shell
docker run [OPTIONS] IMAGE [COMMAND] [ARG...]
```
